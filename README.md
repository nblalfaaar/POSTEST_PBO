# Sistem Manajemen Tanaman Obat Keluarga (TOGA)

## Deskripsi
Program ini adalah aplikasi berbasis teks untuk mengelola data Tanaman Obat Keluarga (TOGA). Program dibuat menggunakan bahasa pemrograman Java dengan menerapkan konsep Object Oriented Programming (OOP), seperti class, object, property, method, dan constructor.

Program dapat melakukan operasi CRUD (Create, Read, Update, Delete) pada data tanaman, manfaat, dan resep secara bersamaan.

---

## Struktur Class
Program ini memiliki 3 class data:
- **Tanaman** : menyimpan nama tanaman dan nama latin
- **Pengguna** : menyimpan penyakit yang disembuhkan dan bagian tanaman yang digunakan
- **Catatan** : menyimpan bahan dan cara pembuatan

---

## Cara Menjalankan
1. Pastikan Java sudah terinstall
2. Buka terminal, masuk ke folder `src/`
3. Kompilasi semua file:
   ```
   javac *.java
   ```
4. Jalankan program:
   ```
   java Main
   ```
